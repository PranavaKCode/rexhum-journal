# Rexhum Journal

A dedicated viewer for Pranava Kumar's Research Experience in Humanities journal entries.

The entries were ported from the public portfolio post folder and are stored locally in `src/data/entries.ts`. The original portfolio repository is not required at runtime.

## Local Development

```bash
yarn
yarn dev
```

## Checks

```bash
yarn lint
yarn build
yarn test
```

## Demo Notes

Show the archive list, search for a theme like `dracula` or `heaven`, open a few entries, hover or tap the inline note icons, and scroll an entry to show the reader progress bar.
